# RP Philip Hue

### Versioning 
  - Expo v38.0.2
  - ReactN v0.61.4
  - NodeJS v10

### Installation

RP Philip Hue requires [Node.js](https://nodejs.org/) v10+ to run.
Please install [Yarn](https://yarnpkg.com/lang/en/) as well.

Install the dependencies and devDependencies and start the metro server.

```sh
$ cd rp-philip-hue-ts
$ yarn install
$ yarn start
```
